## 🎓 Carian Keputusan Pelajar - Dashboard Keputusan Pemilihan Elektif PPISMP Sem III 25/26

Dashboard interaktif untuk pencarian keputusan pemilihan kursus elektif pelajar TESL.

### 📋 Fitur Utama

✅ **Pilih Kelas** - Dropdown untuk memilih kelas (TESL 1, TESL 2, TESL 3)
✅ **Cari Pelajar** - Dropdown untuk memilih nama pelajar berdasarkan kelas
✅ **Lihat Keputusan** - Tampilkan keputusan pemilihan elektif pelajar
✅ **Responsive Design** - Kompatibel dengan desktop dan mobile
✅ **User-Friendly** - Interface yang mudah digunakan

### 📊 Data yang Ditampilkan

- **ID Pelajar** - Nomor identitas pelajar
- **Nama Pelajar** - Nama lengkap pelajar
- **Kelas** - Tingkat kelas pelajar
- **Elektif 1** - Pilihan kursus elektif pertama
- **Elektif 2** - Pilihan kursus elektif kedua

### 🚀 Cara Menggunakan

1. **Buka Dashboard**
   - Buka file `index.html` di browser

2. **Pilih Kelas**
   - Klik dropdown "Pilih Kelas"
   - Pilih salah satu: TESL 1, TESL 2, atau TESL 3

3. **Pilih Pelajar**
   - Dropdown "Pilih Nama Pelajar" akan otomatis diisi dengan daftar pelajar dari kelas yang dipilih
   - Pilih nama pelajar dari dropdown

4. **Cari Keputusan**
   - Klik tombol "🔍 Cari Keputusan"
   - Hasil pencarian akan ditampilkan dengan informasi lengkap pelajar

5. **Tutup Hasil**
   - Klik tombol "Tutup" untuk menutup hasil pencarian
